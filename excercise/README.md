

**Running Application**
```
> git clone https://github.com/CoderFromCasterlyRock/LTSEExcercise.git
> mvn clean install
> java -jar target/excercise-1.0.0-jar-with-dependencies.jar
```

**Running Tests**
``` 
> mvn clean test
```
